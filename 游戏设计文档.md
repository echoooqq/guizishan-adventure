# 《桂子山校园秘境探险》游戏设计文档（GDD）

> **版本**：v1.0  
> **类型**：2D 俯视角 · 校园探索 · 解谜冒险  
> **参考风格**：《星露谷物语》俯视角 + 《塞尔达：缩小帽》场景解谜  
> **技术栈**：Python 3 + Pygame  

---

## 一、游戏概述

### 1.1 核心概念

你是一名刚踏入华中师范大学的新生，在开学报到途中误入校园秘境——一个与现实校园重叠却暗藏玄机的异世界。桂子山上的桂花突然逆季盛开，校园中七个标志性地点被秘境力量笼罩，每个地点都隐藏着一块**「桂花徽章」碎片**。唯有集齐全部七枚碎片，才能解开秘境封印，回到现实。

### 1.2 游戏特色

| 特色 | 说明 |
|------|------|
| 🗺️ 无缝大地图 | 本部校区为一张连续地图，无加载画面自由探索；南湖校区为独立地图，校车过渡切换 |
| 🧩 场景解谜 | 每个地标建筑内含专属谜题，需观察环境线索并互动 |
| 🎒 背包系统 | 收集、储存、组合、使用道具 |
| 💾 自动存档 | 退出即保存，下次从原地继续 |
| 🌙 昼夜循环 | 白天探索校园，夜晚秘境异变，解锁隐藏内容 |
| 🏫 校园还原 | 七大标志性地点以像素风还原华师校园风貌，双校区设定还原真实校园格局 |

### 1.3 游戏流程概览

```
开场动画 → 新生报到（教程） → 自由探索校园 → 发现秘境入口
    → 探索7个地标地点 → 解谜获取7枚桂花徽章碎片
    → 集齐碎片 → 前往喷泉广场 → 最终解谜 → 通关结局
```

## 二、视觉风格

### 2.1 整体美术方向

- **视角**：2D 俯视角（3/4 俯视，与《星露谷物语》一致）
- **画风**：16-bit 像素风，温暖色调为主，校园绿意盎然
- **分辨率**：游戏内部分辨率 480×270，窗口缩放至 960×540 或 1280×720
- **Tile 尺寸**：16×16 像素（基础地块），角色约 16×24 像素

### 2.2 色彩基调

| 时段 | 色调 | 说明 |
|------|------|------|
| 白天 | 暖黄 + 翠绿 | 阳光明媚的校园氛围 |
| 黄昏 | 橘红 + 金色 | 夕阳西下，光影拉长 |
| 夜晚 | 深蓝 + 紫色 | 秘境力量显现，建筑微微发光 |
| 秘境异变 | 幽绿 + 暗紫 | 部分区域被秘境侵蚀时的异常色调 |

### 2.3 角色设计

- **主角**：新生形象，可简单选择男女（仅外观差异），像素小人，4方向行走动画（每方向3帧）
- **NPC**：各场景驻守的校园角色（学长、图书管理员、食堂阿姨等），有简单待机动画

## 三、地图设计

### 3.1 整体地图结构

游戏地图分为**两个独立校区**，对应现实中的华师本部校区与南湖校区：

#### 本部校区（无缝大地图）

本部校区为一张**无缝大地图**，总尺寸约 **120×80 tiles**（1920×1280 像素内部分辨率），摄像机跟随玩家平滑滚动。游戏中除南湖综合楼外的6个标志性地点均位于本部校区，玩家可自由行走于道路、草坪、广场之间，无需加载。

地图分为**室外区域**和**室内区域**两层：

- **室外区域**：无缝连通，玩家可自由行走于桂中路、博雅广场、喷泉广场等室外场景之间
- **室内区域**：进入建筑时切换到室内地图（有简短过渡动画，非加载黑屏），室内地图独立绘制

```

│         │  图书馆   │              │ 博雅广场  │       │

#### 南湖校区（独立地图）

南湖校区为一张**独立地图**，与本部校区通过**校区接驳站**进行场景切换过渡。南湖校区地图总尺寸约 **40×30 tiles**，包含南湖综合楼及周边景观。

```

│    │   南湖综合楼       │       │

│             │                  │

#### 两校区关系

| 属性 | 本部校区 | 南湖校区 |
|------|----------|----------|
| 地图类型 | 无缝大地图 | 独立地图 |
| 包含地点 | 图书馆、博雅广场、桂中路、佑铭体育馆、学子食堂、喷泉广场 | 南湖综合楼 |
| 地图尺寸 | 120×80 tiles | 40×30 tiles |
| 过渡方式 | 内部无缝行走 | 通过校区接驳站切换 |

### 3.2 地图区块详细设计

#### 3.2.1 桂中路（东西主干道）

| 属性 | 内容 |
|------|------|
| 类型 | 室外 · 道路 |
| 尺寸 | 约 60×6 tiles |
| 地形 | 砖石路面 + 两侧桂花树 + 路灯 |
| 功能 | 连接各区域的主干道，也是游戏初始区域 |
| 互动元素 | 桂花树（可摇晃掉落道具）、路灯（夜晚点亮）、长椅（可坐下触发回忆对话） |
| 谜题 | **「桂花密语」**：7棵桂花树中有1棵被秘境标记，在夜晚时微微发光，对其使用「摇树」互动可获取桂花徽章碎片 |
| 隐藏道具 | 桂花枝（可拾取，用于后续组合道具） |

#### 3.2.2 南湖综合楼（南湖校区）

南湖综合楼位于独立的南湖校区地图中，需从本部校区的校区接驳站乘校车前往。

| 属性 | 内容 |
|------|------|
| 类型 | 独立校区 · 室外区域 + 室内多层建筑 |
| 校区室外尺寸 | 40×30 tiles（南湖校区整体地图） |
| 综合楼外观尺寸 | 14×10 tiles（南湖校区地图内） |
| 室内尺寸 | 20×15 tiles × 2层 |
| 地形 | 南湖水面 + 现代教学楼外观 + 滨湖步道 + 桂花林 |
| 功能 | 核心解谜地点之一，南湖校区唯一可探索区域 |
| 互动元素 | 黑板（可查看）、电脑终端（可交互）、书架（可翻阅）、公告栏（线索）、南湖湖畔长椅（可坐下触发对话）、桂花林（可探索） |
| NPC | 神秘学长（综合楼1楼内，提供提示） |
| 谜题 | **「建校之钥」**：在1楼公告栏发现线索"1903年，华师肇始"，需在2楼电脑终端输入年份 **1903** 解锁密室通道，密室中获取桂花徽章碎片 |
| 隐藏道具 | 旧校徽（1楼黑板后暗格）、实验室钥匙（2楼抽屉）、南湖贝壳（滨湖步道，彩蛋道具） |

**南湖校区室外区域细节**：

- **滨湖步道**：沿南湖水面铺设的木质栈道，可远眺湖面，步道尽头有一张长椅，坐下触发主角独白（暗示秘境与南湖的关联）
- **桂花林**：综合楼旁的小片桂花林，白天可拾取桂花瓣（消耗品，恢复10体力），夜晚林中偶尔可见秘境微光
- **校区接驳站**：位于南湖校区地图南端，互动后可选择返回本部校区

### 3.3 地图连接与过渡

| 过渡类型 | 实现方式 | 示例 |
|----------|----------|------|
| 室外↔室外（同校区） | 无缝，摄像机平滑跟随 | 桂中路 → 博雅广场 |
| 室外→室内 | 玩家走到建筑门口按互动键，淡入淡出过渡 | 图书馆门口 → 图书馆1楼 |
| 室内→室外 | 走到出口自动触发，淡入淡出过渡 | 图书馆1楼门口 → 图书馆外 |
| 室内楼层 | 走到楼梯按互动键，简短过渡 | 图书馆1楼 → 图书馆2楼 |
| **跨校区** | 走到校区接驳站按互动键，校车过场动画过渡 | 本部校区接驳站 → 南湖校区 |

#### 跨校区过渡详细设计

- **本部校区接驳站位置**：位于喷泉广场南侧，有一个校车站牌标记
- **南湖校区接驳站位置**：位于南湖校区地图南端
- **过渡动画**：玩家互动后播放像素风校车行驶过场动画（约2-3秒），随后淡入目标校区
- **过渡条件**：首次前往南湖校区需先在本部校区触发相关剧情（如从NPC处听说南湖综合楼的异常），之后可自由往返
- **昼夜不影响**：校车全天运行，任何时段均可往返

### 3.4 小地图

- 屏幕右上角显示简化小地图
- 小地图随当前所在校区自动切换显示
- 已探索区域显示地形轮廓，未探索区域为迷雾
- 当前玩家位置用亮点标注
- 各地标建筑用图标标注
- 跨校区时小地图整体切换，两校区小地图独立维护迷雾状态

## 四、核心玩法系统

### 4.1 移动系统

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | WASD / 方向键 | 8方向移动（上下左右 + 斜向） |
| 冲刺 | Shift + 移动 | 加速移动，消耗体力 |
| 互动 | F / 空格 | 与面前物体/NPC互动 |
| 打开背包 | Tab / I | 打开背包界面 |
| 打开地图 | M | 打开全局地图 |

- 移动速度：步行 2 tiles/秒，冲刺 3.5 tiles/秒
- 碰撞检测：基于 Tile 的碰撞，不可通过地形（墙壁、水面、围栏等）阻挡移动
- 角色朝向：4方向（上下左右），互动时面向目标

### 4.2 互动系统

玩家靠近可互动物体时，物体上方显示**互动提示气泡**（如 "💬 对话"、"🔍 查看"、"✋ 拾取"），按互动键触发：

| 互动类型 | 触发条件 | 效果 |
|----------|----------|------|
| 对话 | 面向NPC | 显示对话框，推进剧情 |
| 查看 | 面向可查看物体 | 显示描述文本，可能获得线索 |
| 拾取 | 面向地面道具 | 道具加入背包 |
| 使用 | 背包中选中道具后面对目标 | 对目标使用道具（如用钥匙开门） |
| 机关操作 | 面向可操作机关 | 触发机关（如踩地砖、操作记分牌） |

### 4.4 谜题系统

#### 4.4.1 谜题设计原则

1. **环境叙事**：线索融入场景，不突兀
2. **渐进引导**：NPC对话提供方向，环境细节提供具体线索
3. **容错设计**：错误尝试有提示反馈，不设死局
4. **校园特色**：谜题内容与华师校史、校园文化相关

#### 4.4.2 谜题总览

| 地点 | 谜题名称 | 类型 | 核心机制 | 难度 |
|------|----------|------|----------|------|
| 桂中路 | 桂花密语 | 观察+时机 | 夜晚找到发光桂花树 | ★☆☆ |
| 南湖综合楼 | 建校之钥 | 知识+输入 | 发现建校年份并输入 | ★★☆ |
| 图书馆 | 书海寻踪 | 问答+排列 | 回答问题+书籍归位 | ★★★ |
| 博雅广场 | 地砖阵法 | 观察+顺序 | 按正确顺序踩踏地砖 | ★★☆ |
| 佑铭体育馆 | 精准一击 | 小游戏+密码 | 投篮+调整记分牌 | ★★☆ |
| 学子食堂 | 饭卡迷踪 | 搜索+对话 | 找到隐藏饭卡并归还 | ★☆☆ |
| 喷泉广场 | 七徽归位 | 综合+排列 | 按正确顺序嵌入7枚碎片 | ★★★ |

#### 4.4.3 谜题状态追踪

每个谜题有独立状态：

```
未发现 → 已发现（获得线索）→ 进行中 → 已解决（获得徽章碎片）
```

- **未发现**：玩家尚未到达该区域或未触发相关对话
- **已发现**：通过NPC对话或环境线索得知谜题存在
- **进行中**：正在解谜，部分条件已满足
- **已解决**：谜题完成，获得桂花徽章碎片

### 8.2 核心技术方案

| 模块 | 方案 | 说明 |
|------|------|------|
| 地图编辑 | Tiled Map Editor | 使用Tiled绘制地图，导出为TMX格式，Pygame通过 `pytmx` 库加载 |
| 地图渲染 | 仅渲染视口内Tile | 摄像机裁剪，只绘制屏幕可见区域，保证性能 |
| 碰撞检测 | Tile-based | 地图编辑器中标记碰撞层，运行时基于Tile坐标判断 |
| 精灵动画 | Sprite Sheet | 每角色一张精灵表，代码按帧切割播放 |
| 昼夜光照 | 全局色调叠加 | 在游戏画面上叠加半透明色彩层，随时间渐变 |
| 对话系统 | 数据驱动 | 对话内容存储在JSON文件，支持条件分支和变量替换 |
| 存档 | JSON序列化 | 游戏状态序列化为JSON文件，支持自动和手动存档 |

### 8.3 性能优化

1. **视口裁剪**：仅渲染摄像机可见范围内的Tile和实体
2. **脏标记渲染**：静态地形仅在变化时重绘，动态实体每帧更新
3. **实体剔除**：远离玩家的NPC/物体不更新逻辑
4. **资源预加载**：进入区域时预加载相关资源，避免卡顿
5. **地图分块**：大地图按Chunk分块加载，按需载入/卸载

## 十一、游戏流程时序

### 11.1 开场（约3分钟）

1. **标题画面**：游戏Logo + "开始游戏 / 继续游戏 / 设置"
2. **开场动画**：像素风过场——新生拖着行李箱走进华师校门
3. **教程引导**：桂中路上，系统提示基本操作（移动、互动、背包）
4. **秘境触发**：走到桂中路中段，桂花突然逆季盛开，画面色调异变，秘境降临

### 11.2 自由探索阶段（核心游玩，约30-60分钟）

- 玩家可自由选择探索顺序（7个地点中6个可任意顺序）
- 本部校区6个地点可无缝步行到达；南湖综合楼需从喷泉广场南侧校区接驳站乘校车前往
- 首次前往南湖校区需先在本部校区触发相关剧情（从NPC处听说南湖综合楼异常），之后可自由往返
- 每个地点：进入 → 探索环境 → 发现线索 → 解谜 → 获得徽章碎片
- 昼夜交替影响可探索内容
- 喷泉广场在集齐6枚碎片前被秘境屏障阻挡

### 11.3 最终阶段（约5-10分钟）

1. 集齐6枚碎片后，喷泉广场屏障解除
2. 进入喷泉广场，与秘境守护者对话
3. 进行「七徽归位」最终解谜
4. 第7枚碎片在正确嵌入前6枚后自动显现
5. 全部嵌入 → 喷泉光芒大作 → 秘境消散
6. 结局动画：校园恢复平静，桂花缓缓飘落

### 11.4 结局差异

| 条件 | 结局 |
|------|------|
| 仅收集7枚徽章通关 | 普通结局：回到现实，校园恢复 |
| 额外完成所有隐藏道具收集 | 完美结局：额外彩蛋动画，秘境守护者真实身份揭晓 |
| 3天内（游戏时间）通关 | 速通结局：特殊称号画面 |
