# 《桂子山校园秘境探险》游戏设计文档（GDD）

> **版本**：v1.0  
> **类型**：2D 俯视角 · 校园探索 · 解谜冒险  
> **参考风格**：《星露谷物语》俯视角 + 《塞尔达：缩小帽》场景解谜  
> **技术栈**：Python 3 + Pygame  

---

## 一、游戏概述

### 1.1 核心概念

你是一名刚踏入华中师范大学的新生，在开学报到途中误入校园秘境——一个与现实校园重叠却暗藏玄机的异世界。桂子山上的桂花突然逆季盛开，校园中七个标志性地点被秘境力量笼罩，每个地点都隐藏着一块**「桂花徽章」碎片**。唯有集齐全部七枚碎片，才能解开秘境封印，回到现实。

### 1.2 游戏特色

| 特色 | 说明 |
|------|------|
| 🗺️ 无缝大地图 | 本部校区为一张连续地图，无加载画面自由探索；南湖校区为独立地图，校车过渡切换 |
| 🧩 场景解谜 | 每个地标建筑内含专属谜题，需观察环境线索并互动 |
| 🎒 背包系统 | 收集、储存、组合、使用道具 |
| 💾 自动存档 | 退出即保存，下次从原地继续 |
| 🌙 昼夜循环 | 白天探索校园，夜晚秘境异变，解锁隐藏内容 |
| 🏫 校园还原 | 七大标志性地点以像素风还原华师校园风貌，双校区设定还原真实校园格局 |

### 1.3 游戏流程概览

```
开场动画 → 新生报到（教程） → 自由探索校园 → 发现秘境入口
    → 探索7个地标地点 → 解谜获取7枚桂花徽章碎片
    → 集齐碎片 → 前往喷泉广场 → 最终解谜 → 通关结局
```

## 二、视觉风格

### 2.1 整体美术方向

- **视角**：2D 俯视角（3/4 俯视，与《星露谷物语》一致）
- **画风**：16-bit 像素风，温暖色调为主，校园绿意盎然
- **分辨率**：游戏内部分辨率 480×270，窗口缩放至 960×540 或 1280×720
- **Tile 尺寸**：16×16 像素（基础地块），角色约 16×24 像素

### 2.2 色彩基调

| 时段 | 色调 | 说明 |
|------|------|------|
| 白天 | 暖黄 + 翠绿 | 阳光明媚的校园氛围 |
| 黄昏 | 橘红 + 金色 | 夕阳西下，光影拉长 |
| 夜晚 | 深蓝 + 紫色 | 秘境力量显现，建筑微微发光 |
| 秘境蛰伏态 | 极淡暖绿 | 白天秘境蛰伏，校园微妙异常感 |
| 秘境觉醒态 | 幽绿 + 深蓝 | 夜晚秘境觉醒，绿色脉冲+路灯发绿光 |

### 2.3 角色设计

- **主角**：新生形象，可简单选择男女（仅外观差异），像素小人，4方向行走动画（每方向3帧）
- **NPC**：各场景驻守的校园角色（学长、图书管理员、食堂阿姨等），有简单待机动画

## 三、地图设计

### 3.1 整体地图结构

游戏地图分为**两个独立校区**，对应现实中的华师本部校区与南湖校区：

#### 本部校区（无缝大地图）

本部校区为一张**无缝大地图**，总尺寸约 **120×80 tiles**（1920×1280 像素内部分辨率），摄像机跟随玩家平滑滚动。游戏中除南湖综合楼外的6个标志性地点均位于本部校区，玩家可自由行走于道路、草坪、广场之间，无需加载。

地图分为**室外区域**和**室内区域**两层：

- **室外区域**：无缝连通，玩家可自由行走于桂中路、博雅广场、喷泉广场等室外场景之间
- **室内区域**：进入建筑时切换到室内地图（有简短过渡动画，非加载黑屏），室内地图独立绘制

```
┌─────────────────────────────────────────────────────┐
│                   本部校区 · 无缝大地图                │
│                     北 · 校园北区                      │
│         ┌──────────┐              ┌──────────┐       │
│         │  图书馆   │              │ 博雅广场  │       │
│         │ (室内地图) │              │ (室外区域) │       │
│         └─────┬────┘              └─────┬────┘       │
│               │     桂中路（东西主干道）    │           │
│  西 ──────────┼──────────────────────────┼───── 东    │
│               │                          │           │
│         ┌─────┴────┐              ┌─────┴────┐       │
│         │佑铭体育馆 │              │ 学子食堂  │       │
│         │ (室内地图) │              │ (室内地图) │       │

│                     南 · 校园南区                      │

│              └──────────┘                             │
└─────────────────────────────────────────────────────┘
```

#### 南湖校区（独立地图）

南湖校区为一张**独立地图**，与本部校区通过**校区接驳站**进行场景切换过渡。南湖校区地图总尺寸约 **40×30 tiles**，包含南湖综合楼及周边景观。

```
┌───────────────────────────────┐
│        南湖校区 · 独立地图       │
│                               │
│    🌊 南湖水面                  │
│    ┌──────────────────┐       │
│    │   南湖综合楼       │       │
│    │   (室内地图×2层)   │       │
│    └────────┬─────────┘       │
│             │                  │
│    🚌 校区接驳站（→ 本部校区）   │
│             │                  │
│    🌿 滨湖步道 + 桂花林         │
│                               │
└───────────────────────────────┘
```

#### 两校区关系

| 属性 | 本部校区 | 南湖校区 |
|------|----------|----------|
| 地图类型 | 无缝大地图 | 独立地图 |
| 包含地点 | 图书馆、博雅广场、桂中路、佑铭体育馆、学子食堂、喷泉广场 | 南湖综合楼 |
| 地图尺寸 | 120×80 tiles | 40×30 tiles |
| 过渡方式 | 内部无缝行走 | 通过校区接驳站切换 |
| 室内区域 | 各建筑独立室内地图 | 综合楼独立室内地图 |

南湖综合楼位于独立的南湖校区地图中，需从本部校区的校区接驳站乘校车前往。

| 属性 | 内容 |

| 综合楼外观尺寸 | 14×10 tiles（南湖校区地图内） |
| 室内尺寸 | 20×15 tiles × 2层 |

- **桂花林**：综合楼旁的小片桂花林，白天可拾取桂花瓣（消耗品，恢复10体力），夜晚林中偶尔可见秘境微光
- **校区接驳站**：位于南湖校区地图南端，互动后可选择返回本部校区

### 3.3 地图连接与过渡

| 过渡类型 | 实现方式 | 示例 |
|----------|----------|------|
| 室外↔室外（同校区） | 无缝，摄像机平滑跟随 | 桂中路 → 博雅广场 |
| 室外→室内 | 玩家走到建筑门口按互动键，淡入淡出过渡 | 图书馆门口 → 图书馆1楼 |
| 室内→室外 | 走到出口自动触发，淡入淡出过渡 | 图书馆1楼门口 → 图书馆外 |
| 室内楼层 | 走到楼梯按互动键，简短过渡 | 图书馆1楼 → 图书馆2楼 |
| **跨校区** | 走到校区接驳站按互动键，校车过场动画过渡 | 本部校区接驳站 → 南湖校区 |

#### 跨校区过渡详细设计

- **本部校区接驳站位置**：位于喷泉广场南侧，有一个校车站牌标记
- **南湖校区接驳站位置**：位于南湖校区地图南端
- **过渡动画**：玩家互动后播放像素风校车行驶过场动画（约2-3秒），随后淡入目标校区
- **过渡条件**：首次前往南湖校区需先在本部校区触发相关剧情（如从NPC处听说南湖综合楼的异常），之后可自由往返
- **昼夜不影响**：校车全天运行，任何时段均可往返

### 3.4 小地图

- 屏幕右上角显示简化小地图
- 小地图随当前所在校区自动切换显示
- 已探索区域显示地形轮廓，未探索区域为迷雾
- 当前玩家位置用亮点标注
- 各地标建筑用图标标注
- 跨校区时小地图整体切换，两校区小地图独立维护迷雾状态

## 四、核心玩法系统

### 4.1 移动系统

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | WASD / 方向键 | 4方向移动（上下左右） |
| 冲刺 | Shift + 移动 | 加速移动，消耗体力 |
| 互动 | F / 空格 | 与面前物体/NPC互动 |
| 打开背包 | Tab / I | 打开背包界面 |
| 打开地图 | M | 打开全局地图 |
| 打开任务日志 | J | 打开任务日志 |
| 打开菜单 | Esc | 暂停菜单 |

- 移动速度：步行 2 tiles/秒，冲刺 3.5 tiles/秒
- 碰撞检测：基于 Tile 的碰撞，分轴检测（先 X 后 Y），允许沿墙滑动
- 角色朝向：4方向（上下左右），互动时面向目标

### 4.2 互动系统

玩家靠近可互动物体时，物体上方显示**互动提示气泡**（中文文本），按互动键触发：

| 互动类型 | 提示文本 | 触发条件 | 效果 |
|----------|---------|----------|------|
| 对话 | "按 F 对话" | 面向NPC | 显示对话框，推进剧情 |
| 查看 | "按 F 查看" | 面向可查看物体 | 显示描述文本，可能获得线索 |
| 拾取 | "按 F 拾取" | 面向地面道具 | 道具加入背包，拾取后物体从场景消失 |
| 使用 | "按 F 使用" | 背包中选中道具后面对目标 | 对目标使用道具（如用钥匙开门） |
| 机关操作 | "按 F 操作" | 面向可操作机关 | 触发机关（如踩地砖、操作记分牌） |
| 进入 | "按 F 进入" | 面向门/入口 | 触发地图过渡（淡入淡出） |
| 切换楼层 | "按 F 切换楼层" | 面向楼梯 | 触发楼层切换 |
| 乘校车 | "按 F 乘校车" | 面向校区接驳站 | 触发跨校区校车过渡 |

> **注意**：当前版本未实现"丢弃后道具出现在地面可再次拾取"功能。丢弃操作仅从背包移除道具。

### 4.4 谜题系统

#### 4.4.1 谜题设计原则

1. **环境叙事**：线索融入场景，不突兀
2. **渐进引导**：NPC对话提供方向，环境细节提供具体线索
3. **容错设计**：错误尝试有提示反馈，不设死局
4. **校园特色**：谜题内容与华师校史、校园文化相关

#### 4.4.2 谜题总览

| 地点 | 谜题名称 | 类型 | 核心机制 | 难度 | 独占界面 |
|------|----------|------|----------|------|----------|
| 桂中路 | 桂花密语 | 观察+时机 | 夜晚找到发光桂花树并摇树 | ★☆☆ | 否（PLAYING状态下完成） |
| 南湖综合楼 | 建校之钥 | 知识+输入 | 发现建校年份并在终端输入 | ★★☆ | 是（绿色终端界面） |
| 图书馆 | 书海寻踪 | 问答 | 回答3道随机题目 | ★★★ | 是（答题界面） |
| 博雅广场 | 地砖阵法 | 观察+顺序 | 按随机顺序踩踏8块地砖 | ★★☆ | 否（PLAYING状态下踩踏） |
| 佑铭体育馆 | 精准一击 | 小游戏+密码 | 投篮小游戏+记分牌拨盘 | ★★☆ | 是（投篮+记分牌界面） |
| 学子食堂 | 饭卡迷踪 | 搜索+对话 | 找到隐藏饭卡并归还 | ★☆☆ | 否（对话+搜索完成） |
| 喷泉广场 | 七徽归位 | 综合+排列 | 按固定顺序嵌入7枚碎片 | ★★★ | 是（凹槽排列界面） |

> **机制简化说明**：图书馆谜题省略了"书籍归位到2楼书架"的后续物理交互；体育馆省略了"获得器材室钥匙→打开器材柜→获得记分牌便条"的前置道具流程；食堂省略了"归还饭卡→打开后厨门→冰箱取徽章"的物理流程；博雅广场地砖由设计的9块固定顺序简化为8块随机顺序；喷泉广场嵌入顺序由设计的"线索推断"简化为固定顺序（badge_1~badge_7）。

#### 4.4.3 谜题状态追踪

每个谜题有独立状态：

```
未发现 → 已发现（获得线索）→ 进行中 → 已解决（获得徽章碎片）
```

- **未发现**：玩家尚未到达该区域或未触发相关对话
- **已发现**：通过NPC对话或环境线索得知谜题存在
- **进行中**：正在解谜，部分条件已满足
- **已解决**：谜题完成，获得桂花徽章碎片

### 8.2 核心技术方案

| 模块 | 方案 | 说明 |
|------|------|------|
| 地图编辑 | Tiled Map Editor | 使用Tiled绘制地图，导出为TMX格式，Pygame通过 `pytmx` 库加载 |
| 地图渲染 | 仅渲染视口内Tile | 摄像机裁剪，只绘制屏幕可见区域，保证性能 |
| 碰撞检测 | Tile-based | 地图编辑器中标记碰撞层，运行时基于Tile坐标判断 |
| 精灵动画 | Sprite Sheet | 每角色一张精灵表，代码按帧切割播放 |
| 昼夜光照 | 全局色调叠加（关键帧插值法） | 11个关键帧线性插值，全天连续渐变，秘境蛰伏/觉醒双态叠加 |
| 对话系统 | 数据驱动 | 对话内容存储在JSON文件，支持条件分支和变量替换 |
| 存档 | JSON序列化 | 游戏状态序列化为JSON文件，支持自动和手动存档 |

### 8.3 性能优化

1. **视口裁剪**：仅渲染摄像机可见范围内的Tile和实体
2. **脏标记渲染**：静态地形仅在变化时重绘，动态实体每帧更新
3. **实体剔除**：远离玩家的NPC/物体不更新逻辑
4. **资源预加载**：进入区域时预加载相关资源，避免卡顿
5. **地图分块**：大地图按Chunk分块加载，按需载入/卸载

## 十一、游戏流程时序

### 11.1 开场（约3分钟）

1. **标题画面**：游戏Logo（从上方滑入动画）+ "开始游戏 / 继续游戏 / 设置"，按 ↑/↓ 选择，回车确认
2. **开场动画**：黑屏文字 → 校门场景 → 秘境降临，共3个阶段
   - 阶段0：黑屏文字"桂子山秘境探险"淡入淡出
   - 阶段1：校门场景，玩家角色从下方走入
   - 阶段2：走到桂中路中段触发秘境降临（幽绿闪烁→白光→恢复）
3. **教程引导**：分步骤弹出操作提示（移动WASD→互动F→拾取→背包Tab→地图M→菜单Esc→冲刺Shift）
4. **秘境触发与守护者对话**：走到桂中路中段触发，守护者现身对话告知秘境已苏醒

### 11.1.1 开场详细流程

1. **校门出生**：开场动画结束后，玩家出生在南端校门内（校门含门柱、横梁、牌匾）
2. **分步教程**：
   - TUTORIAL_INTRO：开场动画
   - TUTORIAL_INTERACT：提示与路过的学生对话（按F）
   - TUTORIAL_MOVE：提示WASD移动
   - TUTORIAL_PICKUP：提示拾取地面水壶（按F）
   - TUTORIAL_INVENTORY：提示按Tab打开背包
   - TUTORIAL_MAP：提示按M打开地图
   - TUTORIAL_MENU：提示按Esc打开菜单
   - TUTORIAL_SPRINT：提示Shift冲刺
   - TUTORIAL_DONE：教程完成
3. **沿路北行**：校门到桂中路的石板路引导玩家向北走
4. **秘境触发**：走到桂中路中段矩形区域时触发秘境降临
5. **异变演出**：幽绿闪烁→白光→恢复，秘境蛰伏态生效
6. **守护者对话**：4句台词告知秘境已苏醒、需集齐碎片、昼夜机制
7. **提示文字**：屏幕中央"探索校园，寻找七个被秘境笼罩的地点"（5秒淡出）
8. **首次夜晚提示**：当游戏时间进入夜晚且秘境已触发时，显示提示"去桂中路看看……"

### 11.2 自由探索阶段（核心游玩，约30-60分钟）

- 玩家可自由选择探索顺序（7个地点中6个可任意顺序）
- 本部校区6个地点可无缝步行到达；南湖综合楼需从喷泉广场南侧校区接驳站乘校车前往
- 首次前往南湖校区时播放 nanhu_intro 对话，之后可自由往返
- 每个地点：进入 → 探索环境 → 发现线索 → 解谜 → 获得徽章碎片
- 昼夜交替影响可探索内容（桂中路谜题仅夜晚可解，广场舞阿姨仅夜晚出现等）
- 喷泉广场在集齐6枚碎片前被秘境屏障阻挡
- 6个引导NPC散布在校园各处，提供操作提示和地点引导
- 解谜后部分引导NPC对话会变化（如还书同学在图书馆解谜后会有新对话）

### 11.3 最终阶段（约5-10分钟）

1. 集齐6枚碎片后，喷泉广场屏障解除
2. 进入喷泉广场，与秘境守护者对话
3. 进行「七徽归位」最终解谜（固定顺序 badge_1~badge_7）
4. 前6枚正确嵌入后，触发第7枚粒子显现动画（粒子汇聚→光芒中显现）
5. 全部嵌入 → 喷泉光芒大作 → 秘境消散 → 触发结局动画

### 11.4 结局差异

| 条件 | 结局 |
|------|------|
| 仅收集7枚徽章通关 | 普通结局（normal）：校园恢复平静 |
| 持有桂花书签和校徽暗纹等彩蛋道具通关 | 好结局（good）：额外旁白，守护者身份暗示 |
| （设计中）3天内通关 | 真结局（true）：特殊称号画面（代码中预留，触发条件需完善） |

> **结局彩蛋系统**：根据玩家持有的彩蛋道具数量（桂花书签、校徽暗纹、博雅石等），结局旁白会有不同深度的追加内容。
