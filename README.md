# 桂子山秘境探险

一款以华中师范大学校园为背景的 2D 俯视角解谜冒险游戏。玩家扮演误入秘境的新生，需在校园中探索、解谜，集齐 7 枚桂花徽章碎片解除封印。

## 游戏简介

- **类型**：2D 俯视角 · 校园探索 · 解谜冒险
- **参考风格**：《星露谷物语》俯视角 + 《塞尔达：缩小帽》场景解谜
- **背景设定**：华中师范大学（华师）校园
- **核心目标**：集齐 7 枚桂花徽章碎片，解除秘境封印

## 环境要求

- **Python**：3.8 或更高版本
- **操作系统**：Windows（推荐）/ macOS / Linux
- **屏幕分辨率**：最低 960×540

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动游戏

**方式一**：双击 `启动游戏.bat`（Windows）

**方式二**：命令行启动

```bash
python main.py
```

## 游戏流程

1. **标题画面** → 按任意键开始
2. **开场动画** → 新生走进校门
3. **教程引导** → 学习基本操作
4. **自由探索** → 在本部校区和南湖校区之间探索
5. **解谜收集** → 完成 7 个谜题，收集 7 枚桂花徽章碎片
6. **七徽归位** → 在喷泉广场完成最终谜题
7. **结局** → 根据通关天数和收集道具触发不同结局

## 七大谜题

| 谜题 | 地点 | 类型 | 难度 |
|------|------|------|------|
| 桂花密语 | 桂中路 | 观察+时机 | ★☆☆ |
| 建校之钥 | 南湖综合楼 | 知识+输入 | ★★☆ |
| 书海寻踪 | 图书馆 | 问答+排列 | ★★★ |
| 地砖阵法 | 博雅广场 | 观察+顺序 | ★★☆ |
| 精准一击 | 佑铭体育馆 | 小游戏+翻牌 | ★★☆ |
| 饭卡迷踪 | 学子食堂 | 搜索+对话 | ★☆☆ |
| 七徽归位 | 喷泉广场 | 综合+排列 | ★★★ |

## 三种结局

| 结局 | 触发条件 |
|------|---------|
| 真结局 | 2 天内通关且收集全部道具 |
| 好结局 | 3 天内通关 |
| 普通结局 | 3 天以上通关 |

## 技术栈

- **Python 3** — 主语言
- **Pygame** — 渲染引擎、输入处理、UI 自定义渲染
- **pygame_gui** — UI 管理器（已初始化，UI 模块采用纯 Pygame 自定义渲染）
- **pytmx** — 加载 Tiled Map Editor 导出的 TMX 地图
- **Tiled Map Editor** — 地图编辑工具
- **Zpix** — 中文像素风字体

## 游戏特色

- **昼夜循环**：游戏内 1 天 = 现实 6 分钟，白天/黄昏/夜晚光照渐变
- **秘境双态**：白天蛰伏态（淡绿异常）与夜晚觉醒态（幽绿脉冲），通关后秘境消散
- **7 大谜题**：涵盖观察、知识、问答、时机、搜索、组合等多种解谜类型
- **3 种结局**：根据通关速度和收集度触发不同结局
- **校区探索**：本部校区 + 南湖校区，校车往返
- **室内外切换**：图书馆、体育馆、食堂、综合楼等室内场景自由进出
